Let's say you know a great piece of open source software that you'd like to host as part of a LoRes Mesh.
{{< /hero >}}

Perhaps this is software you built yourself, or perhaps it's just software that you enjoy using or want to make available to your community. To do that, someone needs to bundle it up as a **LoRes App**, and in this section we'll walk through that journey together.
